# 运行环境

* Window10
* MacOS
* ~win7~ (由于采用最新的qt6技术所以不支持win7)

电视中需要安装接收端例如

* 当贝投屏 无广告
* ~乐播投屏~ 投屏后有广告恶心人
* 银河奇异果 无广告
